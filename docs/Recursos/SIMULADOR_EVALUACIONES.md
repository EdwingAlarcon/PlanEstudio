# 🎯 Simulador de Evaluaciones por Módulo (Estilo Microsoft)

Este simulador replica la dinámica esperada en evaluaciones tipo Microsoft Learn/certificación:

- navegación por preguntas
- selección única y múltiple
- finalización con puntaje
- explicación de respuesta
- reintento por módulo

> Recomendación: aprueba cada módulo con **70% o más** antes de avanzar al siguiente nivel.
